Pour demarrer un environnement virtuel:
source venv/bin/activate 

Installer les dependances du requirements.txt
pip install -r requirements.txt

Demarrer un serveur backend (FastAPI)
uvicorn main:app --reload

envoyer un prompt au LLM via le terminal: 
curl -X POST \
-H "Content-Type: application/json" \
-d '{"prompt": "Salut, présentes toi en 10 mots maximum."}' \
http://localhost:8000/generate \
